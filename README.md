# 만화 번역 오버레이 (Android)

일본 만화/웹소설을 읽을 때, **페이지가 바뀌면 자동으로** 화면 위에 한글을 덧그려 주는 오버레이 앱입니다.

- 온디바이스 OCR(일본어) + 온디바이스 번역(일→한) → **검열 없음, 오프라인 작동, 빠름**
- 말풍선 위치에 흰 배경을 깔고 한글을 **가로(좌→우)** 로 박스에 맞춰 표시
- 작화(그림·의성어)는 건드리지 않고 대사만 교체
- 페이지가 "넘어간 뒤 멈추는 순간"을 감지해 그때 한 번만 번역 → 페이지 단위로 가장 빠르게

---

## 빌드 방법 (APK 만들기)

이 폴더는 **완성된 소스 프로젝트**입니다. APK는 Android Studio에서 만드세요.

1. **Android Studio** 설치 (Hedgehog 이상 권장)
2. `File > Open` → 이 폴더(`MangaOverlayTranslator`) 선택
3. 첫 실행 시 Gradle/SDK가 자동으로 동기화됩니다 (인터넷 필요, 몇 분 소요)
4. 상단 메뉴 `Build > Build App Bundle(s) / APK(s) > Build APK(s)`
5. 완료되면 `app/build/outputs/apk/debug/app-debug.apk` 생성 → 휴대폰에 설치

> 또는 USB로 폰을 연결하고 ▶(Run) 버튼을 누르면 바로 설치·실행됩니다.
> 실기기 권장(에뮬레이터는 화면 캡처가 제한될 수 있음).

---

## 사용 방법

1. 앱 실행 → **번역 시작** 누르기
2. "다른 앱 위에 표시" 권한 허용 → 다시 **번역 시작**
3. 알림 권한 허용 → **화면 캡처 동의** (안드로이드 보안상 세션마다 1회)
4. 자동으로 만화 앱으로 이동. 좌상단에 **번역 ON / 종료** 버튼이 떠 있음
5. 만화를 읽으며 페이지를 넘기면, 멈추는 즉시 한글이 덧그려집니다
6. 끄려면 떠있는 **종료** 버튼 또는 앱의 **번역 종료**

> 최초 1회는 번역 모델(약 30MB)이 다운로드된 뒤부터 동작합니다(이후 오프라인).

## 속도 / 정확도 튜닝

`ScreenCapture.kt` 하단 상수를 조절하세요.

| 상수 | 의미 | 빠르게 ↔ 안정적 |
|---|---|---|
| `STABLE_MS` | 화면이 멈췄다고 판단하는 시간 | 줄이면 더 빠르게 반응(오작동↑) |
| `MOVE_THRESHOLD` | "움직임"으로 보는 변화량 | 낮추면 작은 스크롤도 감지 |
| `PAGE_THRESHOLD` | "새 페이지"로 보는 변화량 | 낮추면 더 자주 재번역 |
| `THROTTLE_MS` | 프레임 처리 간격 | 낮추면 반응↑ 발열↑ |

## 알려진 한계 (솔직하게)

- **화면 캡처 동의**는 안드로이드 규칙상 세션마다 1번 필요합니다(자동 생략 불가).
- ML Kit 온디바이스 번역은 빠르고 검열이 없지만, 품질은 LLM(GPT/Claude)보다 다소 떨어집니다. 더 자연스러운 번역이 필요하면 `TranslatePipeline` 의 번역 부분만 외부 API 호출로 교체하면 됩니다(단, API 정책에 따라 노골적 표현이 필터될 수 있음).
- **세로쓰기·손글씨·작은 의성어 옆 대사**는 인식 정확도가 들쭉날쭉합니다. 더 정밀하게 하려면 말풍선 감지 모델(예: comic-translate의 RT-DETR)을 OCR 앞단에 추가하세요.
- ML Kit/SDK 버전은 시점에 따라 더 최신이 있을 수 있습니다. Android Studio가 권하면 업데이트하세요.

## 구조

```
app/src/main/java/com/example/mangaoverlay/
├─ MainActivity.kt       권한·화면캡처 동의 → 서비스 시작
├─ OverlayService.kt     포그라운드 서비스, 오버레이 창 + ON/OFF 버튼, 전체 흐름 연결
├─ ScreenCapture.kt      화면 캡처 + "페이지 정지 순간" 감지(빠른 페이지 번역의 핵심)
├─ TranslatePipeline.kt  일본어 OCR → 일→한 온디바이스 번역
└─ OverlayView.kt        말풍선 위치에 흰 배경 + 한글(가로, 박스 맞춤) 렌더링
```
